Project Overview

Project Objectives

Redesign the Department of Homeland Security (DHS) website to enhance user experience, accessibility, and information circulation. This includes modernizing the design, improving navigation, and ensuring compliance with web standards and accessibility guidelines.

Research Method Keywords

Heuristic Evaluation, User Interviews, Card Sorting, Usability Testing.

Project Outcomes

The redesigned website will serve as a centralized hub for citizens, stakeholders, and partners to access vital information, resources, and services provided by DHS.
